Selama tujuh pertemuan pertama, saya mengalami perjalanan belajar Data Science yang terstruktur dan bertahap:

Pertemuan 1 menjadi fondasi awal, di mana saya mengenal Python, variabel, list, perulangan, dan fungsi – semua dalam lingkungan Google Colab yang interaktif.

Pada Pertemuan 2, saya mulai bersentuhan dengan dataset nyata (Titanic) dan belajar melakukan eksplorasi data sederhana menggunakan Pandas, seperti filtering, grouping, dan agregasi.
